# 如何分析一个行业
* 行业概况‌
判断行业是否处于风口，位于生命周期的哪个阶段(探索期、成长期、成熟期或衰退期)，评估当前进入是否还有红利
* ‌市场分析‌：
包括市场规模、增长趋势、细分市场等，直接决定盈利天花板

* ‌产品研究‌：
分析产品是否具有发展潜力，是否会面临时代淘汰危机

* ‌竞争格局‌：
了解行业蛋糕如何被分割，评估自己能获得的市场份额

* ‌监管政策‌：
分析国家政策对行业的影响，一切行业都跟着政策走

* 其他影响因素‌：
包括地区、文化、经济、生态、人口等因素

# 常用分析方法
* ‌PEST分析‌：评估宏观环境中的政治(Political)、经济(Economic)、社会(Social)和技术(Technological)因素

* ‌波特五力模型‌：分析行业竞争程度和吸引力的五种力量：供应商议价能力、购买者议价能力、潜在竞争者进入能力、替代品替代能力、同业竞争者竞争强度

* ‌SWOT分析‌：评估企业内部的优势(Strengths)和劣势(Weaknesses)，以及外部的机会(Opportunities)和威胁(Threats)

* ‌历史资料研究法‌：通过分析行业历史数据预测未来趋势

* ‌调查研究法‌：通过问卷、访谈等方式获取行业现状及动态

* ‌比较研究法‌：横向(跨公司、跨地区)或纵向(跨时间段)比较行业数据

# 关键指标
* 市场指标:反映行业整体状况
市场规模、增长率、市场份额

* 财务指标:评估行业盈利能力
净利润率、ROI、现金流	

* 用户指标:衡量用户价值
客户获取成本(CAC)、客户终身价值(CLV)	

* 运营指标:反映运营效率
库存周转率、员工满意度

* 数字指标:适用于互联网行业
DAU(日活跃用户)、MAU(月活跃用户)、留存率

# 数据来源渠道
* ‌政府统计机构‌：如中国国家统计局、美国劳工统计局，提供权威行业数据

* ‌行业协会‌：如中国汽车工业协会，发布特定行业运营数据和技术标准

* ‌市场研究公司‌：艾瑞咨询、尼尔森等，提供定制化行业报告(通常需付费)

* ‌专业数据库‌：Wind、Bloomberg等，整合全球行业数据

* ‌公开互联网资源‌：行业网站、新闻平台、社交媒体等提供的实时数据

* ‌学术数据库‌：CNKI、JSTOR等收录的行业研究论文和数据集

# 实用工具与模板
‌分析工具‌：
* FineBI：自助大数据分析工具，市场占有率第一

* Tableau：全球知名的数据可视化工具

* boardmix：提供SWOT和PEST分析模板，支持AI生成分析

# ‌报告模板‌：
* 行业概述(定义、分类、发展历程)

* 市场分析(规模、增长、细分)

* 竞争格局(主要企业、市场份额)

* 行业环境(政策、技术、成本结构)

* 关键成功因素和未来展望
