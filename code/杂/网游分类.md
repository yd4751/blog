# 按游戏类型分
* RPG: 角色扮演游戏  
强调剧情发展和个人体验，分为日式(回合制)和美式(即时制)	《最终幻想》《仙剑奇侠传》《暗黑破坏神》
* ACT: 动作游戏  
强调操作技巧和反应能力，不追求复杂剧情	《鬼泣》《魂斗罗》
* AVG: 冒险游戏  
以解谜和探索为核心，强调故事情节	《神秘岛》
* AAG: 动作冒险游戏  
结合动作和冒险元素，自成一体	《生化危机》《古墓丽影》
* SLG: 策略游戏  
强调策略运用，分为回合制和即时制	《三国志》《星际争霸》
* MOBA: 多人在线战术竞技游戏  
团队对抗，强调战术配合	《英雄联盟》《DOTA2》
* FPS: 第一人称射击游戏  
第一人称视角的射击体验	《使命召唤》《CS》
* TPS: 第三人称射击游戏  
第三人称视角的射击体验	《绝地求生》《战争机器》
